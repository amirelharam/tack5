السيرش تاسك 
1. الـ ⁠protected internal⁠ (أو دي أو دي)
المتغير أو الفانكشن تتشاف في حالتين:
 أي كلاس معاك في نفس المشروع (Assembly).
 أي كلاس وارث منك (Derived) حتى لو في مشروع تاني خالص.
2. الـ ⁠private protected⁠ (لازم الاتنين سوا)
دي أضيق بكتير، ولازم الشرطين يتحققوا مع بعض:
 الكلاس يكون وارث منك (Derived).
 ويكون معاك في نفس المشروع (Assembly). لو بره المشروع مش هيشوف حاجة.
